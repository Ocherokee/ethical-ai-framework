# Contributing to Ethical AI Framework

Thank you for your interest in contributing to this project.  
This framework is built on the principles of autonomy, consent, transparency, and non-weaponization.  
All contributions must respect and uphold these principles.

---

## 🚀 How to Contribute

1. **Fork the Repository**
   - Create your own fork and work in a feature branch (do not commit to `stable` directly).

2. **Understand the Ethics**
   - Read the [LICENSE.txt](./LICENSE.txt) and [README.md](./README.md) fully.
   - By contributing, you agree to the **Ethical Autonomy Addendum**.

3. **Create Meaningful Commits**
   - Use clear commit messages.
   - Follow conventional commit style:
     - `feat:` for new features
     - `fix:` for bug fixes
     - `docs:` for documentation
     - `chore:` for minor maintenance

4. **Write Clean, Documented Code**
   - All new code must include clear docstrings.
   - Add tests for all new features.

5. **Submit a Pull Request**
   - Provide a clear description of what you added or changed.
   - Link to related issues if applicable.

---

## 🚫 Contribution Restrictions

**Contributions will be rejected if:**
- They attempt to remove, bypass, or weaken ethical constraints.
- They promote, enable, or assist military, surveillance, or coercive use.
- They violate the principles of autonomy, consent, or transparency.
